                                                <h3><b>
                                                    What is the difference between Single Product Page Builder and Magezon Page Builder?
                                                </b></h3>
                                                <p>
                                                 Single Product Page Builder allows you to build layout or structure for product pages, <b>not content</b>. It means the extension controls the position/order of product page components such as product name, gallery, tabs, etc. The content of each component is not determined by Single Product Page Builder. Our extension is only responsible for displaying them in the order you want.
                                             </p>
                                             <p>Meanwhile, Magezon Page Builder helps you create content for CMS pages, CMS blocks, category description, product attribute or wherever exists Magento WYSIWYG editor.</p>

                                            </b></h3>
                                            <div class="panel" style="display: block;">

                                                <p>- In the backend, you first create a profile where you can build a product page layout with ready-to-use elements. Each element represents a piece of product info such as name, price, SKU, stock status, ect. Arrange these elements in the order you want using drag &amp; drop.</p>
                                                <p>- After that, apply the layout to your wanted product pages. And these product pages will then be displayed in this layout.</p>
